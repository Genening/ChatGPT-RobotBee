# ChatGPT RobotBee
![image](https://user-images.githubusercontent.com/37908189/225677418-f8c8afb6-0747-426e-80a5-446d2b51fb68.png)


参考源仓库：[ChatGPT_JCM](https://github.com/202252197/ChatGPT_JCM)

## 下载依赖包
```
npm install
```

## 运行
```
npm run serve
```

## 编译
```
npm run build
```

## electron预览
```
npm run start
```

## electron打包可执行exe文件
```
npm run packager
```
